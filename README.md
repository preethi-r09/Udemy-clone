# Udemy-clone

Udemy Clone - Frontend Design
A high-fidelity frontend clone of the Udemy learning platform. This project focuses on modern UI/UX design, featuring a responsive layout, interactive navigation, and categorized course sections.

‚ú® Features
1. Navigation Bar
Logo: Branding link to the homepage.
Search Bar: Interactive input field for course discovery.
User Links: Login, Signup, and Cart icons with hover effects.
Responsive Menu: Mobile-friendly hamburger menu for smaller screens.

2. Categories Section
A tabbed or grid-based navigation allowing users to browse specific fields: 
Development: Web, Mobile, and Game Dev.
Design: UI/UX, Graphic Design, and Illustration.
Marketing: SEO, Social Media, and Digital Marketing.
IT & Software: Cyber Security, Cloud Computing, and Certifications.

3. Recommended Section
A curated list of top-rated courses featuring:
Course thumbnails and titles.
Instructor names and star ratings.
Price tags and "Bestseller" badges.

4. Topics Recommended Section
A "pill-shaped" badge layout showcasing trending tags like:
JavaScript, Python, Photoshop, React, Data Science, AWS.

5. Most Popular Section
A high-traffic carousel or grid showcasing the most-enrolled courses across all categories globally.
